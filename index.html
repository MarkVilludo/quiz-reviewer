1. What does the computer software acronym JVM stand for?

The Java Virtual Machine (JVM) is a virtual machine that enables a computer to run Java programs as well as programs written in other languages that are also compiled to Java bytecode.

The JVM is a crucial component of the Java ecosystem, allowing Java code to run on any platform that has a JVM implementation, without the need for recompilation.

Answer: Java Virtual Machine

11. How many bits make up the significand portion of a single precision floating point number?

In IEEE 754 single-precision floating-point format, the significand (or mantissa) has 23 bits. Together with the sign bit and exponent, this makes up a 32-bit representation.

Floating-point numbers are a fundamental data type in computing, and understanding their representation is crucial for many applications, including scientific computing and graphics.

Answer: 23 bits

12. Unix Time is defined as the number of seconds that have elapsed since when?

Unix Time is a widely used timestamp format, and its origin is a significant milestone in the history of computing.

Answer: January 1, 1970 (midnight UTC/GMT)

19. The acronym "RIP" stands for which of these?

"RIP" stands for Routing Information Protocol. It is a distance-vector routing protocol used in local and wide area networks to help routers determine the best path for data packets.

RIP is one of the oldest routing protocols and is used in various networking scenarios. It operates based on hop count and has limitations in scalability and efficiency compared to more modern protocols.

Answer: Routing Information Protocol

22. According to DeMorgan's Theorem, the Boolean expression (AB)' is equivalent to:

DeMorgan's Theorem states that the complement of a conjunction is the disjunction of the complements. Therefore, (AB)' is equivalent to A' + B'.

DeMorgan's Theorem is a fundamental principle in Boolean algebra, used to simplify expressions and understand the relationships between different logical operations.

Answer: A' + B'

23. How many bytes are in a single Kibibyte?

A single Kibibyte (KiB) is equal to 1024 bytes. This is based on the binary system, where 1 Kibibyte = 210 bytes.

Kibibytes are part of the binary prefix system, which is used in computing to denote data sizes. It differs from the metric system, where a kilobyte (KB) is 1000 bytes.
